This program is mainly a demo to showcase my custom IMGUI and hardware accellerated renderer inspired by the RAD Debugger. It does not work well for painting and it cannot save any files you paint (although it does work really well as a photo viewer). It can, however, read Photoshop brushes in a primitive manner. It also accepts pressure sensitive input from a drawing tablet or Cintiq. I'm unable to build it anymore, but the code itself is useful.
